# Proyecto completo de Teoría de Códigos

Incluye:
- Fundamentos con tarjetas animadas y modal.
- Punto I Reed–Solomon sobre F5 con algoritmo paso a paso.
- Punto II código lineal binario y operaciones con algoritmo paso a paso.
- Punto III código cíclico binario con algoritmo paso a paso.
- Sección dinámica de Códigos Reed–Solomon.
- Sección dinámica de Códigos Lineales.
- Sección dinámica de Códigos Cíclicos.

Ejecutar:
```bash
npm install
npm run clean
npm run dev
```
